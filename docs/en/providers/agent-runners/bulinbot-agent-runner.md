# Built-in Agent Runner

By default, BulinBot uses the built-in Agent Runner as the default executor. You don't need to configure anything to use BulinBot's powerful built-in Agent Runner.

![image](https://files.bulinbot.app/docs/source/images/bulinbot-agent-runner/image.png)

With the built-in Agent Runner, you can use BulinBot's [MCP Server](/use/mcp), [Knowledge Base](/use/knowledge-base), [Web Search](/use/websearch), and persona features.
